Conceptos clave

Para entender y explicarle Doofinder al cliente durante la llamada

Store
Representa el sitio web del cliente en Doofinder. Una cuenta puede tener varias stores para distintos países o idiomas.
Search Engine
Motor de búsqueda que indexa el catálogo. Puede haber varios por idioma o tipo de catálogo dentro de la misma Store.
Data Feed
Archivo XML, CSV o JSON con todos los productos. Doofinder lo lee periódicamente para mantener el catálogo actualizado.
Indexación
Proceso por el que Doofinder lee el feed y guarda los productos. La primera puede tardar varios minutos según el tamaño del catálogo.
Layer
Widget visual que aparece superpuesto en la web cuando alguien escribe en el buscador. Muestra resultados en tiempo real.
Script de instalación
Una única línea de JavaScript que conecta la tienda con Doofinder. Va en el header o footer de todas las páginas.
CSS Selector
Indica a Doofinder dónde está el buscador. Ej: #search, input[name="q"], .search-input
Plugin vs Script manual
El plugin (Shopify, WooCommerce…) automatiza todo: feed, script, selector. El script manual requiere pegar el código en el HTML.
Script de instalación
<script src="https://eu1-config.doofinder.com/2.x/STORE_ID.js" async></script>
eu1 = Europa us1 = América ap1 = Asia · STORE_ID en Admin Panel → Configuración → Store Settings
